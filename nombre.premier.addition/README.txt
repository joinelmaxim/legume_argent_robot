READ ME

1) You need to have the JAVA-SE12 (jdk.12.0.2) installed on your computer

2) Apache Maven 3.6.1 has to be installed on your computer too.

3) Sts (Spring Tool Suite) is the last software that you must install.

4) Create a workspace with Sts

5) Click on File> Import > Maven > Existing Maven Projects. Click "Next". Click "Browse". Select the project. Click "Finish".

If you want to run the application:

6)Right click on the project Run as > Java Application

Click on "com.joinel.test2.adneom.nombre.premier.addition"

Test if you want to run the tests:

6)Right Click on the project the project Run > JUNIT Test
